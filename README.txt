== Description ==

WIP Incoming Lite is a free Coming Soon, Under Construction & Maintenance Mode WordPress plugin and allows you to manage a launch / under construction page for your WordPress website

**Premium features.**

= Installation =

Once you have installed the plugin, you just need to activate the plugin in order to enable it.

= Configuration =

WIP Incoming Lite will add a new admin page, where you can customize the login page.

== Installation ==

1. Upload the plugin files to the '/wp-content/plugins/wip-incoming-lite' directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to ‘WIP Incoming' page to configure the plugin.
